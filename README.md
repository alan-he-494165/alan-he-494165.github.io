# alan-he-494165.github.io

Personal blog powered by Jekyll and GitHub Pages.

## Run locally

1. Install Ruby + Bundler
2. Install dependencies:
   bundle install
3. Run local server:
   bundle exec jekyll serve
4. Open:
   http://127.0.0.1:4000

## Publish

Push commits to `main` branch.
GitHub Pages will build and publish automatically at:
https://alan-he-494165.github.io
